Alexander OS - безопасная версия для публикации на GitHub Pages

Важно:
- В исходном коде нет личных балансов, имён клиентов и ожидаемых оплат.
- Данные, которые вы введёте после запуска, сохраняются локально в браузере устройства.
- Не храните в приложении пароли, API-ключи, паспортные данные и другие секреты.
- Для публикации загрузите СОДЕРЖИМОЕ этой папки в корень репозитория.
