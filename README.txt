Alexander OS v3 - обновление

Что изменено:
- Общий капитал считается как сумма всех счетов, наличных, накоплений и инвестиций минус открытые долги.
- Ликвидные деньги и свободный остаток после ближайших платежей считаются отдельно.
- Текущая неделя и месяц сравниваются с теми же прошедшими днями прошлого периода.
- Можно выбрать любые два месяца и сравнить доход, расходы, зарплату, сбережения и категории.
- Во вкладке «Прогресс» собрана динамика денег, капитала, задач, привычек, проектов и целей.
- Добавлена автоматическая фиксация ежедневных показателей для истории капитала.
- Главная упрощена до командного центра: предупреждения, финансовый пульс, 3 задачи и ближайшие деньги.
- Расходы можно помечать как обязательные/необязательные и личные/рабочие, а операции связывать с проектами.
- Кнопки «Отмена» и крестик в формах закрывают окно.
- Существующие данные из прошлой версии сохраняются.

Установка обновления:
1. Сделайте экспорт резервной копии в настройках приложения.
2. Распакуйте архив.
3. В GitHub откройте репозиторий alexander-os.
4. Нажмите Add file -> Upload files.
5. Загрузите все файлы из этой папки в корень репозитория.
6. Commit message: Update Alexander OS v3.
7. Нажмите Commit changes.
8. Подождите публикацию GitHub Pages и полностью перезапустите приложение.
