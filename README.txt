Alexander OS V11.3 - Neon Lime & UI Fix

Изменения:
- Добавлена новая тема «Неон лайм» с тёмно-графитовыми карточками и кислотным акцентом.
- Исправлено открытие настроек через профиль.
- Настройки теперь открываются в аккуратной модальной панели с фиксированным заголовком и независимой прокруткой.
- Кнопка закрытия остаётся доступной во время прокрутки.
- Увеличен отступ между графиком общего прогресса и карточками показателей.
- Иконка «Привычки» заменена на аккуратную векторную иконку.
- Текущие иконки нижнего меню, порядок вкладок и логика V11 сохранены.

Загрузка в GitHub Pages:
1. Распакуйте архив.
2. Загрузите все файлы в корень репозитория с заменой старых.
3. Дождитесь завершения GitHub Actions / Pages.
4. Полностью закройте установленное приложение и откройте снова.

Перед обновлением рекомендуется создать резервную копию данных в настройках.
