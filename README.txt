Alexander OS v10.0 - GitHub Pages

Что внутри:
- 6 вкладок: Главная, Финансы, Задачи, Проекты, Прогресс, Настройки
- новый дизайн v10 с тремя темами
- быстрый расход через верхнюю кнопку +
- доходы, расходы, проекты, цели, привычки и заметки
- локальная блокировка PIN и Face ID через WebAuthn
- автоблокировка при сворачивании и бездействии
- обычный JSON-экспорт и защищённая .aos-копия AES-GCM
- импорт JSON и .aos с полным восстановлением данных
- экспорт отчёта для ChatGPT

Загрузка на GitHub:
1. Распакуйте архив.
2. Загрузите все файлы в корень репозитория с заменой старых.
3. Дождитесь завершения GitHub Pages.
4. Полностью закройте приложение на iPhone и откройте снова.

Важно:
- Face ID работает только через HTTPS на поддерживаемом устройстве.
- Перед обновлением сделайте резервную копию данных.
- Защищённая копия .aos не восстанавливается без заданного пароля.
