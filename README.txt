Alexander OS v7 - Liquid Glass update

Что изменилось:
- Новый цельный дизайн в стиле будущего iOS / Liquid Glass.
- Внизу 6 аккуратных вкладок: Сегодня, Задачи, Финансы, Проекты, Прогресс, Настройки.
- Проекты и Прогресс сдвинуты левее, Настройки добавлены последней вкладкой.
- Кнопка быстрого добавления осталась только наверху справа.
- Убраны дублирующие плавающие кнопки внутри экранов.
- Настройки открываются из нижнего меню.
- Обновлены карточки, формы, модальные окна, вкладки и навигация.
- Сохранена выгрузка полного отчёта для ChatGPT.

Как обновить:
1. Сначала сделайте резервную копию в приложении.
2. Распакуйте архив.
3. Загрузите все файлы в корень репозитория GitHub с заменой старых.
4. Commit changes.
5. Дождитесь успешного Pages deployment.

Данные остаются в локальном хранилище браузера. Не очищайте Safari до резервного копирования.
