Alexander OS v2

Обновление через GitHub:
1. Откройте репозиторий alexander-os.
2. Add file -> Upload files.
3. Загрузите все файлы из этой папки с заменой старых.
4. Commit changes напрямую в main.
5. Подождите 1-3 минуты и обновите приложение.

Данные сохраняются под прежним ключом localStorage и автоматически переносятся из v1.
Перед обновлением рекомендуется сделать экспорт резервной копии в настройках приложения.
