Alexander OS V12.2 - сборка для GitHub Pages

Что исправлено:
- Нижнее меню собрано заново как ровный iPhone tab bar из 5 одинаковых секций.
- Убраны визуальные пустоты, старые grid-column, двойной нижний отступ и конфликтующие размеры.
- В портретной ориентации меню занимает аккуратную ширину экрана.
- В горизонтальной ориентации меню ограничено по ширине и не растягивает пункты по всему экрану.
- Все иконки и подписи имеют одинаковые размеры, вертикальное выравнивание и отступы.
- Плавающая кнопка + появляется, когда в нижнюю часть экрана входит последний содержательный блок.
- Дотягивать страницу до абсолютного низа больше не нужно.
- При прокрутке обратно вверх кнопка исчезает у третьего содержательного блока.
- Логика работает на Главной, в Финансах, Задачах, Проектах и Прогрессе.
- В Life Map плавающая кнопка скрыта, потому что там есть отдельная верхняя кнопка добавления.
- Верхняя кнопка + не изменена.
- Обновлён Service Worker и номер кэша, чтобы iPhone не показывал старую версию.

Как обновить:
1. Сохрани резервную копию через экспорт данных в Alexander OS.
2. Удали старые файлы из корня репозитория GitHub.
3. Загрузи все файлы из этого архива в корень репозитория.
4. Дождись обновления GitHub Pages.
5. Полностью закрой установленное веб-приложение и открой его снова.

Данные localStorage сохранятся, если адрес GitHub Pages не меняется.
