Alexander OS v10.4 Exact Design

Готовая PWA-версия для GitHub Pages.

Включено:
- 6 вкладок в утвержденном премиальном дизайне
- Изумрудная, графитовая и светлая темы
- Финансы, счета, платежи и быстрый расход
- Задачи, заметки, проекты, цели, привычки и динамика
- Face ID / WebAuthn и резервный PIN
- Автоблокировка
- Экспорт JSON и защищенный .aos
- Точное восстановление данных
- Экспорт отчета для ChatGPT
