Alexander OS v4

Главные изменения:
- Любой доход и расход автоматически меняет баланс выбранного счёта.
- Для старых операций без счёта создаётся «Основной баланс».
- Общий капитал и вкладка «Прогресс» пересчитываются сразу.
- Сравнение месяцев перенесено в компактное модальное окно.
- Единый тёмный дизайн с лаймовым акцентом на всех экранах.

Перед обновлением сделайте экспорт резервной копии.
