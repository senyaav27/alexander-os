Alexander OS v9.0 - GitHub Pages release

Основные изменения:
- Финансы собраны в одну большую зелёную карточку без повторяющихся KPI.
- В карточке: капитал, активы, долги, ликвидность, доход месяца, зарплата,
  клиенты и проекты, расходы месяца, расходы недели и свободный остаток.
- Проектный доход автоматически создаёт финансовую операцию.
- Прогресс показывает динамику капитала, доходов, расходов, целей, задач и привычек.
- Настройки стали отдельным полноценным экраном.
- Темы: Изумрудная, Графитовая и Светлая.
- Верхняя кнопка + открывает быстрый ввод расхода.
- База мыслей и идей находится во вкладке Задачи.
- Экспорт для ChatGPT сохранён.
- Старые локальные данные совместимы с новой версией.

Публикация:
1. Распаковать архив.
2. Загрузить все файлы в корень репозитория GitHub.
3. Сделать Commit changes.
4. Дождаться Pages build and deployment.
5. Полностью закрыть приложение на iPhone и открыть снова.

Важно:
- Не удалять данные Safari до создания резервной копии.
- Не хранить в приложении пароли, номера карт и документы.
