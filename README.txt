Alexander OS V11.5 - Video Workouts

Что обновлено:
- собственные категории расходов в быстром расходе;
- ключевые слова для автоматического распознавания;
- улучшенный разбор фраз: «кредит 1300», «1 300 продукты», «1,5к кофе»;
- приложение запоминает выбранные категории и формулировки;
- собственные категории доступны в обычных операциях и регулярных платежах;
- категории входят в экспорт, импорт и резервную копию;
- удаление своей категории безопасно переносит старые записи в «Другое».

Загрузка в GitHub:
1. Распакуйте архив.
2. Загрузите все файлы в корень репозитория с заменой.
3. Дождитесь завершения GitHub Pages Actions.
4. Полностью закройте приложение на iPhone и откройте снова.
