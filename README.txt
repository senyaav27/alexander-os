Alexander OS V11.2 - Stability & Intelligence

Готовая PWA-сборка для GitHub Pages.

Добавлено в V11.2:
- история изменений и отмена последнего действия
- корзина с восстановлением удалённых записей
- диагностика базы и встроенные автоматические тесты
- предупреждение о похожих операциях
- регулярные платежи и поступления без двойного учёта
- быстрый ввод расхода одной строкой, например «550 обед»
- финансовый календарь
- журнал тренировок
- автоматический недельный разбор

Сохранено:
- дизайн Alexander OS V11
- Главная, Финансы, Задачи, Проекты, Прогресс, Настройки
- тренировки, цели, привычки и база мыслей
- темы Изумрудная, Графитовая и Светлая
- экспорт/импорт JSON и защищённых копий .aos
- локальная защита PIN и WebAuthn/Face ID на поддерживаемом iPhone

Загрузка: распакуйте архив и загрузите все файлы в корень репозитория GitHub Pages.


V11.2: вкладка «Настройки» убрана из нижнего меню. Настройки открываются по нажатию на профиль/логотип в шапке и закрываются крестиком или нажатием вне окна. Остальные разделы и логика не изменены.
