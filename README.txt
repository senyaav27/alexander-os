Alexander OS V12.1 - исправленная сборка для GitHub Pages

Что исправлено:
- Нижнее меню снова состоит ровно из 5 вкладок: Главная, Финансы, Задачи, Проекты, Прогресс.
- Life Map полностью убран из нижнего меню.
- Life Map открывается кнопкой «Открыть карту жизни» в самом низу раздела «Задачи».
- На экране Life Map добавлены аккуратная шапка, возврат в задачи, ровная сетка статистики и компактные блоки.
- Блоки Life Map, записи внутри них и карточки привычек можно менять местами стрелками на iPhone.
- Перетаскивание мышью сохранено для компьютера.
- Привычки теперь работают по календарным неделям: понедельник - воскресенье.
- Каждая новая неделя начинается автоматически, а старые отметки остаются в истории.
- В каждой привычке есть кнопка истории с динамикой за 10 недель.
- Плавающая кнопка + появляется только у конца страницы и исчезает после прокрутки вверх.
- Обновлён Service Worker, чтобы удалить старый кэш с двумя вкладками Life.

Как установить:
1. Сделай экспорт данных в настройках Alexander OS.
2. Удали старые файлы из корня репозитория GitHub.
3. Загрузи все файлы из этого архива в корень репозитория.
4. Дождись публикации GitHub Pages.
5. Полностью закрой установленное веб-приложение и открой его снова.

Данные хранятся в localStorage браузера и сохранятся при обновлении файлов на том же адресе GitHub Pages.
