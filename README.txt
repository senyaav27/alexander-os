Alexander OS vNext 10.3 - GitHub Pages

Что изменено:
- полностью обновлён визуальный слой по утверждённому макету
- новые SVG-иконки нижней навигации
- отдельная плавающая кнопка + и верхняя кнопка +
- быстрое меню: расход, доход, задача, проект, заметка и счёт
- новый режим темы Future 2030 в дополнение к Изумрудной, Графитовой и Светлой
- полноэкранные нижние формы на iPhone
- новый экран Face ID с автоматическим запуском проверки при открытии
- Face ID можно включить отдельно, без обязательного PIN-кода
- если Face ID и PIN не настроены, приложение открывается без блокировки
- JSON и защищённый .aos экспорт/импорт сохранены
- данные предыдущих версий не сбрасываются

Загрузка на GitHub:
1. Перед обновлением экспортируйте резервную копию.
2. Распакуйте архив.
3. Загрузите все файлы в корень репозитория с заменой старых.
4. Дождитесь завершения GitHub Pages.
5. Полностью закройте Alexander OS на iPhone и откройте снова.

Face ID:
- работает только через HTTPS и встроенный аутентификатор поддерживаемого устройства
- приложение запрашивает только внутренний аутентификатор устройства
- если Safari не разрешит автоматический системный запрос без пользовательского действия, на экране останется кнопка «Повторить Face ID»
